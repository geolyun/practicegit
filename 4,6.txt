- CSS font(글꼴) 속성

1. font-family 글꼴 집합
generic family : 비슷한 모양을 가지는 글꼴 집합 ("Serif", "Monospace" 등)
font family : 특정 글꼴 집합 ("Times", "Courier" 등)

2. font-style 이탤릭체 사용
normal : 텍스트에 어떠한 스타일도 적용하지 않음
italic : 텍스트를 이탤릭체로 나타냄
oblique : 텍스트를 기울임

3. font-variant 
속성값이 small-caps로 설정되면, 텍스트에 포함된 영문자 중 모든 소문자를 작은 대문자(small-caps) 글꼴로 변경

4. font-weight 텍스트 두께

5. font-size 텍스트 크기
1) 절대적 크기 - 텍스트의 크기를 명시된 크기 그대로 설정할 때 사용
2) 상대적 크기 - 텍스트를 둘러싸고 있는 HTML 요소들의 크기에 따라 텍스트의 크기도 변하는 방식

- CSS list-style 속성

1. list-style-type
리스트에 다양한 마커 적용 가능

2. list-style-image
마커로 자신의 이미지 사용 가능

3. list-style-position
리스트 요소의 위치 설정 가능

- CSS 이미지 스프라이트
이미지 스프라이트 : 여러 개의 이미지를 하나의 이미지로 합쳐서 관리하는 이미지

- CSS 크기 단위
가장 많이 쓰이는 단위 : 백분율 단위(%), 배수 단위(em), 픽셀 단위(px)

- 박스 모델
모든 HTML 요소는 박스 모양으로 구성되며, 이것을 박스 모델이라고 부름
내용(content) : 텍스트나 이미지가 들어있는 박스의 실질적인 내용 부분입니다.
패딩(padding) : 내용과 테두리 사이의 간격입니다. 패딩은 눈에 보이지 않습니다.
테두리(border) : 내용와 패딩 주변을 감싸는 테두리입니다.
마진(margin) : 테두리와 이웃하는 요소 사이의 간격입니다. 마진은 눈에 보이지 않습니다.

padding 축약 표현 할때에는 top, right, bottom, left 순으로 설정

-블록과 인라인-
HTML 모든 요소는 display 속성을 가지는데 그 2가지가 블록과 인라인이다.

디스플레이 속성값이 블록인 요소는 언제나 새로운 라인에서 시작, 해당 라인의 모든 너비를 차지
●(<p>, <div>, <h>, <ul>, <ol>, <form>요소는 display 속성값이 블록(block)인 대표적인 요소입니다.)

그 중 가장 대표적인 블록 요소는 <div>요소 → 여러 요소들의 스타일을 한 번에 적용하기 위해 사용

디스플레이 속성값이 인라인인 요소는 새로운 라인에서 시작하지 않음. 요소의 너비도 해당 라인 전체가 아닌 해당 HTML 요소의 내용만큼만 차지
●(<span>, <a>, <img>요소는 display 속성값이 인라인(inline)인 대표적인 요소입니다.)

그 중 가장 대표적인 인라인 요소는 <span>요소 → 주로 텍스트의 특정 부분에 따로 스타일을 적용하기 위해 사용

-iframe 요소-(inline frame)
이 요소를 이용하면 해당 웹 페이지 안에 어떠한 제한 없이 또 다른 하나의 웹페이지를 삽입할 수 있음
문법: <iframe src="삽입할페이지주소"></iframe>

iframe 요소의 페이지 변경
<a>태그를 이용하면 iframe 요소의 최초 페이지를 변경할 수 있음
(<a>태그의 target 속성과 iframe 요소의 name 속성을 연결하면 변경 가능)

프레임셋 - 하나의 브라우저 창에 둘 이상의 페이지를 표시할 수 있음.
프레임셋에서 각각 페이지는 frame 요소로 표현
+(frameset, frame, noframes 요소는 더는 HTML5 표준 권고안에서 지원하지 않습니다.
따라서 하나의 브라우저 창에 여러 페이지를 표현하고 싶을 때는 iframe 요소를 사용하거나 CSS를 이용하여 표현해야 합니다.)

HTML 레이아웃
레이아웃이란 특정 공간에 여러 구성 요소를 보기 좋게 효과적으로 배치하는 작업
HTML에서는 3가지 방법으로 작성 가능
1. div 요소를 이용한 레이아웃
2. HTML5 레이아웃
3. table 요소를 이용한 레이아웃 - 현재 거의 사용 x

Form 요소
form 요소를 사용해 사용자로부터 입력을 받을 수 있고 사용자가 입력한 데이터를 서버로 보낼 때에도 form 요소를 사용
문법: <form action="처리할페이지주소" method="get|post"></form>

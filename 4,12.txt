내비게이션 바
사용자가 웹 사이트에서 가장 많이 클릭하는 영역 중 하나가 바로 내비게이션 바
내비게이션 바 : 우리가 흔히 사용하는 웹 사이트의 메뉴

수직 내비게이션 바
링크를 사용한 리스트 메뉴에 display 속성값을 block으로 설정하면 만들 수 있다.

수평 내비게이션 바
-display 속성의 inline 속성값을 이용한 방법
리스트 메뉴에서 <li>요소의 display 속성값을 inline으로 설정

-floating 속성을 이용한 방법
리스트 메뉴의 <li>요소에 float 속성을 설정
이때 float 속성값을 left로 설정하면, 모든 메뉴가 왼쪽으로 정렬됩니다.
또한, float 속성값을 right로 설정하면, 모든 메뉴가 오른쪽으로 정렬됩니다.

드롭다운 효과
해당 요소에 마우스를 올려서 다른 요소나 텍스트가 나타나게 하는 효과

display:none과 visibility:hidden의 차이
display는 보이지도 않고 해당 공간도 존재하지 않음
visibility는 보이지만 않고 해당 공간은 존재함.

툴팁 효과
해당 요소에 마우스를 올리면 추가적인 정보가 나타나게 하는 효과

width 속성을 이용해 input 요소의 크기를 설정할 수 있다.
border-radius 속성을 이용해 input 요소의 모서리를 둥글게 만들 수 있다.
color 속성을 이용해 input 요소의 텍스트 색상을 변경할 수 있다.

그래디언트란 둘 이상의 색 사이의 색상 표현을 부드럽게 전환해주는 효과

background-size 속성값
contain : 이미지 비율은 유지하면서, 크기를 가능한 한 크게 조절
cover : 이미지 비율은 유지하면서, 요소의 모든 영역을 가리도록 크기를 조절

background-origin 속성, baclground-clip 속성
1. border-box : 배경 이미지를 테두리(border) 영역의 왼쪽 위에 맞춥니다.
2. padding-box : 기본 설정이며, 배경 이미지를 패딩(padding) 영역의 왼쪽 위에 맞춥니다.
3. content-box : 배경 이미지를 내용(content) 영역의 왼쪽 위에 맞춥니다.

border-radius - 모서리를 둥글게 만들 수 있음.

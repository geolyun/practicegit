*선택자

결합 선택자
-자손 선택자 : 해당 요소의 하위 요소 중에서 특정 타입의 요소를 모두 선택
ex) div p {스타일;}
-자식 선택자 : 해당 요소의 바로 밑에 존재하는 하위 요소 중에서 특정 타입의 요소를 모두 선택
ex) div > p {스타일;}

동위 선택자
-일반 동위 선택자 : 해당 요소와 동위 관계에 있으며, 해당 요소보다 뒤에 존재하는 특정 타입의 요소를 모두 선택
ex) div ~ p {스타일;}
-인접 동위 선택자 : 해당 요소와 동위 관계에 있으며, 해당 요소의 바로 뒤에 존재하는 특정 타입의 요소를 모두 선택
ex) div + p {스타일;}

*의사 클래스
문법 : 선택자:의사클래스 이름 {속성: 속성값;}

의사 요소
문법 : 선택자::의사요소이름 {속성: 속성값;}
ex) ::first-letter, ::first-line, ::before, ::after, ::selection

속성 선택자
[속성이름] 선택자, [속성이름="속성값"] 선택자

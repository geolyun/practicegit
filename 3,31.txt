HTML → HyperText Markup Language


<html> HTML 문서의 루트 요소 정의

<head> HTML 문서의 메타데이터 정의
메타데이터: HTML 문서에 대한 정보, 웹 브라우저에는 직접적으로 표현되지 않은 정보

<title> HTML 문서의 제목을 정의

<body> 웹 브라우저를 통해 보이는 내용

<h1>~<h6> 제목

<p> 단락

html 속성은 시작 태그 내에서 속성 이름과 속성값으로 표현

-단락-

<br>태그: 줄 나누기
<pre>태그: 텍스트 서식 미리 정의
<hr>태그: 수평 가로 구분선

-서식-

<b>, <strong> 텍스트를 굵게 표현
<i>, <em> 이탤릭체 표현
<mark> 하이라이팅 표현
<del> 텍스트를 지운 것 같은 효과
<ins> 텍스트를 삽입한 것 같은 효과
<sup> 위첨자 
<sub> 아래첨자

-인용구-

<q> 짧은 인용구
<blockquote> 블록 인용구
<abbr> 축약형 표현
<address> 주소 표현

엔티티: HTML 예약어를 기존에 사용하던 의미 그대로 사용하기 위해 별도로 만든 문자셋
엔티티 문법 ▶ &엔티티이름; or &#엔티티숫자;

문자셋은 html 문서를 나타내기 위해 알아야 할 정보
<head>태그 내의 <meta>태그에 명시
문자셋의 종류: 1. ASCII 2. ANSI 3. ISO-8859-1 4. UTF-8

HTML 스타일
문법: <태그이름 style="속성이름:속성값">
배경색 변경: <h1 style="background-color:white">
글자색 변경: <h1 style="color:maroon">
글자 크기 변경: <h1 style="font-size:250%">
문단 정렬 변경: <h1 style="text-align:center">

HTML 링크
문법: <a href="링크주소">HTML 링크</a>

HTML 이미지
문법: <img src="이미지주소" alt="대체문자열">
src 속성은 이미지가 저장된 URL 주소를 명시
alt 속성은 이미지가 로딩될 수 없는 상황에서 이미지 대신 나타날 문자열을 설정
//이미지 맵 만들기// 
<map>태그를 이용해서 이미지맵 제작 가능
<map>태그는 하나 이상의 <area>태그를 가짐

HTML 리스트
1. 순서가 없는 리스트
<ul>태그로 시작, 각각의 리스트 요소는 <li>태그로 시작
2. 순서가 있는 리스트
<ol>태그로 시작, 각각의 리스트 요소는 <li>태그로 시작
3. 정의 리스트
<dl>태그로 시작, <dt>태그에는 용어의 이름, <dd>태그에는 해당 용어의 대한 정의

HTML 테이블
＊테이블이란 여러 종류의 데이터를 보기 좋게 정리하여 보여주는 표
<tr> 테이블 열 구분	<th> 각 열의 제목		<td> 테이블의 열을 각각의 셀로 나누어줌
border 속성 이용해서 테두리 표현
한줄로 설정하려면 border-collapse로 써야함
column으로 테이블의 열 합치기
